Loan Approval Prediction

Project Overview

This project predicts whether a loan application will be approved using Machine Learning techniques.

Dataset

The dataset contains applicant information such as:

- Married Status
- Education
- Self Employment
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area

Data Preprocessing

- Removed duplicate records
- Handled missing values
- Encoded categorical features
- Removed unnecessary columns

Models Used

1. Logistic Regression
2. Random Forest Classifier
3. Random Forest with Hyperparameter Tuning

Evaluation Metrics

- Accuracy
- Precision
- Recall
- ROC-AUC Score

Feature Importance

The most important feature was Credit_History followed by ApplicantIncome and LoanAmount.

User Prediction

The project also allows users to enter loan details and predict whether the loan will be approved.

Technologies Used

- Python
- Pandas
- Scikit-Learn
- NumPy

Results

Best model achieved approximately:

- Accuracy: 79%
- Recall: 97%
- Precision: 77%
- ROC-AUC: 77%
